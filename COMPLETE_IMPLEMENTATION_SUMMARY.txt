================================================================================
  HOW IT WORKS NOW
================================================================================

VOICE INPUT FLOW:
  User clicks 🎤 Voice
    → Speech Recognition initialized
    → User speaks ingredients
    → Real-time transcript shown
    → Final transcript analyzed
    → Allergen warnings displayed
    → Fallback text input available

IMAGE OCR FLOW:
  User clicks 🖼️ Image
    → File upload dialog opens
    → Image previewed
    → User clicks "Extract Text"
    → Tesseract.js processes (progress shown)
    → Extracted text displayed for editing
    → User confirms
    → Ingredients analyzed
    → Allergen warnings shown

BARCODE SCANNER FLOW:
  User clicks 📷 Barcode
    → Camera access requested
    → QuaggaJS initializes (or fallback)
    → Real-time barcode detection
    → Barcode detected → Open Food Facts lookup
    → Product data returned
    → Ingredients analyzed
    → Allergen warnings shown

TEXT INPUT FLOW:
  User clicks ⌨️ Text
    → Text input field shown
    → User types ingredients
    → User presses Enter
    → Ingredients analyzed
    → Allergen warnings shown
