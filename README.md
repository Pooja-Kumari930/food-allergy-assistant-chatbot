# Food Allergy Assistant Chatbot

This project is a chatbot-based system that helps users detect food allergens.

## Features
- Text input
- Voice input
- Barcode scanning
- Image upload (OCR + detection)

## Technologies Used
- HTML, CSS, JavaScript
- QuaggaJS
- Tesseract.js
- Open Food Facts API
- Web Speech API

## Purpose
To help users identify allergens and make safe food choices.
